# Дизайн и пользовательский интерфейс

Визуальная концепция ByteAway призвана вызвать ощущение премиального, технологичного и безопасного продукта. Приложение и веб-интерфейсы построены с использованием современных дизайн-систем, анимаций и принципов удобства использования (UX).

---

## Концепция интерфейса (UI)

Основной акцент в оформлении сделан на стиле **Glassmorphism** (эффект матового стекла), адаптированном под правила **Material Design 3**.

### Основные элементы стиля
- **Цветовая палитра**: Преобладают темные и глубокие оттенки синего, фиолетового и графитового. Акцентные цвета (ярко-бирюзовый, неоновый зеленый) используются для выделения интерактивных элементов (кнопки подключения, индикаторы статуса).
- **Типографика**: Используются современные геометрические шрифты без засечек (San Francisco, Inter), обеспечивающие высокую читаемость на мобильных экранах любого разрешения.
- **Матовое стекло (Glassmorphism)**: Панели управления, карточки статистики и навигационные элементы имеют полупрозрачный фон с размытием заднего плана (`BackdropFilter`). Это создает ощущение глубины и многослойности интерфейса.

## Пользовательский опыт (UX)

### 1. Процесс онбординга (Onboarding)
- При первом запуске пользователю предлагается краткая анимированная презентация возможностей платформы.
- Процесс регистрации и входа максимально упрощен: требуется ввести только основные данные или использовать токен.

### 2. Главный экран управления
Центральным элементом является интерактивная кнопка активации VPN и Ноды.
- **Состояния кнопки**:
  - *Отключено*: Спокойный контурный стиль с приглушенной подсветкой.
  - *Подключение*: Плавная циклическая анимация вращения градиента вокруг кнопки.
  - *Подключено*: Яркое неоновое свечение, информирующее о защищенности трафика.
- Экран также содержит информационные карточки с текущей скоростью приема и передачи данных, временем сессии и объемом использованного трафика.

### 3. Раздельное туннелирование (Split-Tunneling)
Интерфейс выбора приложений для исключения из VPN выполнен в виде наглядного списка с возможностью быстрого поиска.
- Пользователь может в один клик исключить системные приложения или мессенджеры из туннеля.
- Само приложение ByteAway заблокировано для исключения из списка, чтобы избежать случайных разрывов соединения.

### 4. Веб-интерфейс для B2B (Dashboard)
Веб-интерфейс выполнен в едином визуальном стиле с мобильным приложением.
- Позволяет B2B-клиентам в реальном времени отслеживать остаток баланса и историю платежей.
- Предоставляет инструмент для генерации API-ключей и настройки доступов к прокси-серверам в несколько кликов.
