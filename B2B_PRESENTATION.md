## 🔧 **Техническая интеграция**

### 🐍 **Python (самый популярный)**
```python
from byteaway import ByteAwayClient

# Инициализация
client = ByteAwayClient("your_api_key")

# Настройка прокси
client.setup_proxy("RU-wifi")

# Использование с requests
response = requests.get("https://example.com")

# Использование с selenium
driver = client.get_selenium_driver("US-mobile")
```

### 🌐 **JavaScript/Node.js**
```javascript
const { ByteAwayClient } = require('byteaway');

const client = new ByteAwayClient('your_api_key');
await client.setupProxy('DE-all');

const response = await fetch('https://example.com');
```

### ☕ **Java**
```java
ByteAwayClient client = new ByteAwayClient("your_api_key");
client.setupProxy("GB-wifi");

// Использование с HttpClient
HttpResponse response = client.getHttpClient()
    .execute(new HttpGet("https://example.com"));
```
